# Legacy Archiv
Alte, aber immer noch aktuelle Version des Archives.

Damit der Sppaß läuft, braucht man einen Webserver, eine MySQL DB und ein ```cranach.config.json``` im Wurzelverzeichnis.

```

{
  "db": {
      "host": "localhost",
      "db":   "",
      "user": "",
      "password": ""
  },
  "host": {
    "protocol": "http",
    "hostname": "localhost",
    "port": 8888
  },
  "imagehost": {
    "protocol": "https",
    "hostname": "lucascranach.org",
    "port": false
  }

}
